## Viewing the app

1. Extract the zip
2. Open the file index.html in you favourite browser


## Libraries Used

1. Angular
I believe the app to be a single page application. Angular at present is one of the better frameworks to create SPA's.

2. Lo Dash
Lo dash is a wonderful utility library for JS.

2. Foundation CSS Framework
Foundation is a strong CSS framework which helps to create responsive layouts out of the box.
